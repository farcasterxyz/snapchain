# What is Snapchain?

Snapchain is the decentralized, peer-to-peer network that powers the [Farcaster social network](http://www.farcaster.xyz/). 

![inbound](/images/logo.png)

1. Learn about Snapchain by reading the [whitepaper](/whitepaper) or watching the [video overview](https://www.youtube.com/playlist?list=PL0eq1PLf6eUete1EtUvh7NTIhYCNcR46l).

2. Run a [Snapchain node](guides/running-a-node) to get real-time access to Farcaster data or to help decentralize the network.
